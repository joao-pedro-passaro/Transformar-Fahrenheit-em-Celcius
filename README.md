# Transformar-Fahrenheit-em-Celcius
Transformador rápido de Fahrenheit em Celcius. Informe qualquer temperatura.

Descrição: 
O código usa o valor informado e, apartir da formula, o transfoma em Celcius. 
Obs.: O valor de saída em Celcius está limitado a dois decimais para evitar números muito longos. Ele foi limitado pelo "Celcius:.2f" na formula.
